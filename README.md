# IBM Data Science Portfolio

## Overview

This repository contains projects, notebooks and scripts developed as part of the **IBM Data Science learning track**.

The work demonstrates practical data science skills including:

* Data analysis and exploratory data analysis (EDA)
* Data visualization
* Interactive dashboards
* Python-based data processing
* Applied machine learning workflows

These projects apply real-world datasets to extract insights, visualize trends and communicate findings using industry-standard tools.
